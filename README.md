# Chameleon Ultra 2 

Всем привет и это проет Chameleon ultra 2 на Ардуино! Я решил сделать брелок который может копировать nfc/rfid метки также добавил функцию ик для выключения телевизоров покачто прошивка
разрабатываеться и улучшается.

## Показ зарядки.

В прошивку добавлена индикация для показа заряда батареи с помощью делителя напряжения схема снизу.

### Смена языка

В прошивке будет два языка английский стандартный и руский его можно будет переключать в настройках.

#### Меню

В главном меню будут вкладки
• pn532 
• IR remote controller
• web serwer
• settings
• version 1.3
• Также индикатор заряда в правом верхнем углу.

##### Про IR 

В меню будет такая функция как ик она может выключать телевизоры такие как Sumsung, LG, Panasonic И.тд.

###### Про прошивку

Прошивка будет разделена на папки специально для удобства обновления и упралвения будут папки display, IR, Wifi, pn532, battery, и тд.

####### О функционале

Пока что функционала не много но прошивка будет обновляться и не будет стоять на месте если вы хотите добавить какие-то функции пишитл в тг канал.
